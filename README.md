# simple-rawSocket-synFlood

# 环境和依赖

系统环境：linux

C++版本：C++11 或者更高

GCC版本：GCC4.9 或者更高

库依赖：socket

# 使用

自行编译🤣

sudo ./simple-rawSocket-synFlood.cpp [src_ip] [src_port] [attack_times]

# 例子

sudo ./simple-rawSocket-synFlood.cpp 192.168.1.2 80 5

>SYN攻击192.168.1.2的80端口5次
